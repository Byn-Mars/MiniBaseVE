# MiniBase Vanilla Edition

This is a vanilla only updated version of the [MiniBase: Forever Edition mod](https://github.com/isam2k/MiniBaseSO) which in turn is an adaptation of the famous[MiniBase: Spaced Out Edition mod](https://github.com/antoine-fresse/MiniBaseSO) which in turn is an adaptation of the famous [MiniBase Mod](https://github.com/Versepelles/Oxygen-Not-Included-Mods).
